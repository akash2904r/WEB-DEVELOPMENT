I am an example of outline
The yellow colored part is till where the border exists
The magneta colored line beyond the black line is the outline. It was sent away from the border of the element using the property 'outline-offset'
The rounded edge of the border was given using the property 'border-radius'
Usually it is not possible for the outline to have a rounded corner but, as the outline has inherited it from border, it also shows the same property
