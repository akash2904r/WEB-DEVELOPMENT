Structure of HTML

#1 HTML contains two types of tags:

a) Self closing tags -> Contains only opening tag, there are no closing tags present. For example, tag

b) Pair tags -> Contains an opening as well as closing tag. For example, head tag, p tag (paragraph tag) etc.
